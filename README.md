# Hercules Codes

Displays the 4-symbol level password for every stage directly in the Steam Deck Quick Access menu — no more alt-tabbing to look them up.

## Features

- Three difficulty tabs selectable via icons in the panel header: Beginner, Medium, and Herculian
- Each level's password is shown as four similar to the game symbol images (archer, centaur, coin, gladiator helmet, Hercules, lightning bolt, Medusa, minotaur, Pegasus, serpent, soldier)
- Scrollable level list per difficulty
